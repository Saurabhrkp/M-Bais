# *bin*

*www*

This file is used to start Node.js [*http server*](https://nodejs.org/dist/latest-v12.x/docs/api/http.html) with [*Express*](https://expressjs.com/) app options.

It Normalize Port, start server, when file is called by

``` npm run start ```

or

``` npm run test ```.
